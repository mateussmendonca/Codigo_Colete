# Colete Inteligente
Aqui, adicionamos as informações a respeito do projeto de um colete inteligente que localiza obstáculos a curtas distâncias visando a auxiliar pessoas com deficiência visual em sua locomoção cotidiana.

Para implementar o projeto, serão necessários:
ESP32 (1x), HC-SR04 (3x), Power Bank (1x), Jumper (20x), Buzzer (2x), Botões (2x), Resistor 1k, Protoboard (1x)
